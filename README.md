# Smart-traffic-system

This project aims to develop a smart traffic monitoring system that focuses on vehicle counting and rider safety enforcement. The system detects and counts the number of vehicles in real time to analyze traffic flow and density on roads. Additionally, it identifies two-wheeler riders and determines whether they are wearing helmets or not, helping in enforcing traffic safety rules.
A deep learning-based object detection model such as YOLO26n is applied to detect different types of vehicles and identify helmet usage from video data captured through surveillance cameras. The system processes live or recorded video streams and generates outputs such as total vehicle count and helmet violation alerts.
